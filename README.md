# Pushkar Portfolio V2

Dual-mode portfolio built with Astro and React islands.

- `/` chooses between the two portfolio modes.
- `/formal` is the premium editorial dossier.
- `/formal/cases/[id]` renders static case-study pages from JSON.
- `/xp` mounts the interactive Windows XP-style React portfolio.
- `/contact` exposes direct hiring/contact actions.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Smoke Test

```bash
npm run test:smoke
```
